# 给你的第1个生日宇宙 - 项目规格

## 项目目标

这是一个送给女朋友的生日网页，名称为《给你的第1个生日宇宙》。第一版目标是做成一个手机端优先、结构清晰、可本地运行、后续可部署到 GitHub Pages 或 Vercel 的纵向单页网页。

项目强调浪漫、温柔、精致的生日宇宙氛围。动画可以有，但必须克制、顺滑，不要廉价闪烁、满屏乱飞或影响正文阅读。

## 技术限制

- 使用 HTML + CSS + JavaScript。
- 不使用 React。
- 不使用后端。
- 不使用数据库。
- 不使用登录系统。
- 留言只保存在当前浏览器的 `localStorage`。
- 主要文件是 `index.html`、`style.css`、`script.js`。
- 当前另有 `meteor.js` 和 `meteor-demo.html`，只用于流星背景逻辑和流星效果测试。
- 后续修改必须小步迭代，禁止为了单个问题整体重构页面结构或功能逻辑。

## 页面模块

1. 生日倒计时开场页
   - 生日日期：`2026-06-18T00:00:00+08:00`。
   - 生日结束时间：`2026-06-19T00:00:00+08:00`。
   - 生日前显示“距离你的生日宇宙开启还有”与倒计时。
   - 正式模式下，生日倒计时结束前不能进入完整内容。
   - 生日当天和生日之后可以进入完整内容。
   - `DEV_UNLOCK = true` 仅用于开发测试，生日前可以通过“测试进入生日宇宙”提前测试。

2. 进入生日宇宙首屏
   - 点击进入按钮后，开场模块散开退场。
   - `For My Favorite Star` 首屏作为全屏接替层直接出现在当前视口。
   - 首屏英文小标题、主标题、正文依次划入。
   - 动画结束后页面恢复正常纵向滚动。

3. 回忆时间线
   - 纵向时间线。
   - 每个节点包含日期、标题、文字、照片占位。
   - 当前内容是示例占位，后续替换真实内容。

4. “我眼中的你”互动卡片
   - 多张卡片。
   - 点击后翻转。
   - 当前内容是示例占位。

5. 小游戏 / 小彩蛋
   - 先抽生日签。
   - 抽完后显示记忆问答。
   - 答对后解锁隐藏告白。

6. 最后一页生日信
   - 信纸风格。
   - 当前内容是示例占位。

7. 情绪留言盒
   - 可以写留言。
   - 使用 `localStorage` 保存。
   - 可以查看历史留言。
   - 可以删除留言。
   - 不需要云端同步。

## 设计风格

- 手机端优先。
- 星空、宇宙、生日氛围。
- 标题使用温柔手写风：`Ma Shan Zheng`。
- 正文使用清爽中文无衬线：`Noto Sans SC` 等。
- 英文、数字、按钮、日期、编号使用圆润字体：`Quicksand`。
- 标题和关键文案需要避免“掉字”和奇怪断行。
- 首页标题固定为两行：
  - `给你的第1个`
  - `生日宇宙`
- 开场文案固定分两行，避免孤字。

## 动画原则

- 开场到内容切换可以有明显仪式感。
- 背景流星效果目前仍在调试，先不要作为稳定完成项看待。
- 动画必须支持 `prefers-reduced-motion: reduce`。
- 动画不能影响按钮点击、页面滚动和正文阅读。

## 当前已知状态

- 倒计时逻辑已经实现三状态：生日前、生日当天、生日之后。
- 进入切换动画已改成 staging 全屏接替模式。
- 背景流星已拆到 `meteor.js`，但视觉效果仍需要继续调试。
- `meteor-demo.html` 是独立测试页，不是正式页面模块。

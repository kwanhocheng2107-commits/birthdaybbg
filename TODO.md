## 固定约束复核

- 技术栈固定为 HTML + CSS + JavaScript。
- 不使用 React、不使用后端、不使用数据库。
- 手机端优先。
- 正式模式下，生日倒计时结束后才能进入完整内容；`DEV_UNLOCK` 只用于开发测试。
- 小游戏流程固定为：抽生日签 -> 记忆问答 -> 答对后解锁隐藏告白。
- 情绪留言盒固定使用 `localStorage`，不做云端同步。
- 后续修改只能小步迭代，不能整体重构。
- 入口转场当前固定为：倒计时页退场 -> 第二页淡入，不要恢复两个页面同时叠加显示。
- 月亮是全局背景元素，不要再放回 `.gate` 里随倒计时页隐藏。
- 不要恢复旧的 `.site-main.universe-staging`、`.site-main.universe-entered` 或 `html.universe-transitioning` 机制。
